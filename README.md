# React Toggle Logic Playground

A beginner-friendly React project focused on mastering toggle-based UI logic using useState.

## Features

- 🌙 Dark / Light Mode
- 🔐 Login / Logout Toggle
- 📂 Sidebar Open / Close
- 🪟 Modal Toggle
- 👁 Show / Hide Password
- ❤️ Like Button
- ⭐ Favorite Button
- 📖 Accordion (Coming Soon)
- 📋 Dropdown (Coming Soon)
- 🔔 Notification Panel (Coming Soon)

## Concepts Practiced

- React Components
- useState
- Event Handling
- Conditional Rendering
- Boolean State Management

## Tech Stack

- React
- Vite
- Tailwind CSS

## Goal

This project is part of my React Logic Mastery roadmap where I practice UI logic before building production-level MERN applications.
